# 🎵 מחולל סרטוני שירים לילדים

אפליקציית Python שמקבלת קובץ שמע (שיר), נותנת ל-Gemini *להאזין* לו בפועל דרך ה-File API, בונה ממנו סטורי בורד מבוסס מילים/קצב/מבנה אמיתי, ומייצרת **סרטון MP4 עם אנימציית וידאו אמיתית** באמצעות Veo 3.1 — מסונכרן לאורך השיר.

---

## איך זה עובד?

1. **ניתוח אודיו אמיתי** — קובץ השמע מועלה ל-Gemini דרך ה-File API, ו-Gemini *מאזין* לשיר בפועל (לא רק לשם שלו) ומחלץ ממנו מילים, BPM, מצב רוח, ומבנה סצנות מדויק (זמני התחלה/סיום אמיתיים של כל קטע). אם הניתוח נכשל — נופלים לסטורי בורד גנרי המבוסס רק על שם השיר, עם אורכי סצנה אחידים.
2. **וידאו לכל סצנה** — כל תיאור סצנה נשלח ל-**Veo 3.1** שמייצר קליפ וידאו קצר עם תנועה אמיתית (לא תמונה סטטית). אפשר להעלות **תמונת דמות ייחוס** אחת כדי ש-Veo ישמור על אותה דמות (צבעים, צורה) בכל הסצנות.
3. **הרכבה** — הקליפים מחוברים זה לזה, מותאמים לאורך המדויק של כל סצנה כפי שזוהתה בניתוח האודיו, ומולבש עליהם קובץ האודיו המקורי.
4. **כתוביות קריוקי** — אם לשיר יש מילים, Gemini גם מחזיר תזמון מדויק (שורה + זמן התחלה/סיום) שממנו נבנה קובץ SRT ונשרף לתוך הווידאו עם FFmpeg (טקסט צהוב מודגש עם קונטור שחור עבה, לקריאות מקסימלית לילדים). אפשר לכבות את זה בטופס אם לא רוצים כתוביות.
5. **גיבוי אוטומטי** — אם Veo נכשל בסצנה מסוימת (למשל בגלל מכסה/quota), הסצנה הזו נופלת בחזרה לתמונה סטטית (Imagen 3) עם אפקט Ken Burns (זום ופאן), כך שהסרטון תמיד מוכן. ללא מפתח API בכלל — יש אנימציה פרוצדורלית גנרית (רקע גרדיינט, כוכבים, בועות, אקולייזר).

## הוראות התקנה

### 1. התקן Python

הורד מ-[python.org](https://www.python.org/downloads/) (גרסה 3.10+).
**חשוב:** סמן "Add Python to PATH" בזמן ההתקנה.

### 2. שכפל / פתח את התיקייה

```bash
cd audioToVideo
```

### 3. התקן תלויות

```bash
pip install -r requirements.txt
```

### 4. הגדר מפתח Gemini (אופציונלי אך מומלץ)

צור קובץ `.env` בתיקייה הראשית:

```
GEMINI_API_KEY=your_api_key_here
```

קבל מפתח ב-[Google AI Studio](https://aistudio.google.com/).

> ללא מפתח — האפליקציה תשתמש באנימציה פרוצדורלית גנרית בלבד (ללא Gemini/Veo).

> **חשוב לגבי Veo 3.1:** יצירת וידאו דרך Veo היא פיצ'ר בתשלום/preview עם מכסות (quota) והיא **איטית** — כל סצנה יכולה לקחת 1-3 דקות, כך שסרטון עם 6 סצנות עשוי לקחת עד 15-20 דקות להיווצר. ודא שהמפתח שלך מורשה לגישה ל-`veo-3.1-generate-preview` לפני שימוש מסחרי.

## הפעלה

```bash
python app.py
```

פתח דפדפן בכתובת: **http://localhost:5000**

---

## שימוש

1. גרור קובץ שמע (mp3 / wav / ogg / flac / m4a) לאזור ההעלאה
2. הכנס שם לשיר
3. (אופציונלי) העלה תמונת דמות ייחוס אחת לשמירה על עקביות חזותית
4. השאר מסומן (או בטל) את "הוסף כתוביות קריוקי מסונכרנות"
5. לחץ **"צור סרטון"**
6. המתן (Gemini מאזין לשיר, Veo מייצר וידאו לכל סצנה, ואז נשרפות כתוביות אם יש מילים — כמה דקות)
7. הורד את הסרטון

## מבנה הפרויקט

```
audioToVideo/
├── app.py              # שרת Flask + תזמור ה-pipeline
├── gemini_client.py    # ניתוח אודיו אמיתי + סטורי בורד + תמונות (Imagen 3)
├── veo_client.py       # יצירת קליפי וידאו דרך Veo 3.1
├── animation.py        # Ken Burns + אנימציה פרוצדורלית
├── video_generator.py  # הרכבת הסרטון הסופי וסנכרון אודיו
├── subtitles.py        # בניית SRT + שריפת כתוביות קריוקי עם FFmpeg
├── requirements.txt
├── .env.example
├── templates/
│   └── index.html
└── static/
    ├── css/style.css
    └── js/app.js
```

## דרישות מערכת

- Python 3.10+
- FFmpeg (מגיע אוטומטית עם `imageio-ffmpeg`)
